## About

VisionLab is a modern image annotation and visual labelling platform designed for computer vision and machine learning projects.

The platform supports fast and efficient dataset generation using multiple annotation formats including polygons, bounding boxes, masks, points, and lines.

Built using Python and Qt, VisionLab focuses on scalability, simplicity, and AI-assisted annotation workflows.

---

## Features

* AI-assisted image labelling
* Bounding box annotation
* Polygon and mask segmentation
* Circle, line, and point annotation
* JSON export support
* Multi-image annotation workflow
* Dataset generation for machine learning
* Python + Qt based graphical interface
* Easy dataset management
* Computer vision workflow support
